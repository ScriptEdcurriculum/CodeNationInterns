What I Worked On

During this internship, I worked on building a Slack Bot. Slack Bot is a type of Slack App designed to interact with users via conversation. It is also a simple way to interact with apps and get work done right from your team chat. I built a Slack Bot named Toolkit Bot" and it is built with Slack Real Time Message API (RTM API) and Web API.

Toolkit Bot is connected to Etsy's internal API, it will be able to search the API with a given parameter and return the search results in the conversation. It has three search modes. The default search returns top result from Version One and Version Two of the Toolkit. The second mode is adding "all" to the search, which returns all results from both Versions. The third mode can indicate the number of results returned from the search by simply adding "//num" at the end of the search.
