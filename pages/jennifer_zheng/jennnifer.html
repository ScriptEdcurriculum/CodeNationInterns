My Internship Experience

On the first day of the internship, my manager made me feel welcome. I thought the company would be very strict, but actually, it's very easygoing and enjoyable. In the HSBC tech company, I'm the youngest in my team so everyone treated me friendly and always asked if I need help. I spent my first week there learning what my team is, what it does and how to code with excel VBA since the language is different from what we use at ScriptEd classes. When they introduce Microsoft excel to me, I was surprised since I used Microsoft Excel very often to grid data, but I never know that you can code using Microsoft Excel.

I experienced using Microsoft for one week and then start working on the project my manager gave me. The project was difficult, but my colleague had assisted me with how the basic works and gave me confidence. She supported me and say I can do it. With my teammate's support, I was able to finish the project during my fourth week of the internship at HSBC. I have shown my manager what I accomplished and explained to my manager what I did and how I code it. My manager was very pleased that I finish my project and had gladly asked what I wanted to do in my last week. He said he can ask people from other teams to explain to me what they do if I'm interested. I told him that I'm interested in website designing, so he spends the last week showing me the HSBC websites and teaching me more about HTML, CSS, and javascript.

What I Worked On

During the first week of the internship, I'm learning how to code with Excel VBA by practicing it on the laptop and reading articles about how beginners use Excel VBA and what are the basic functions of VBA programming? After the training, I worked on a project that creates functions for the stock market team. The functions created are used to calculate the data they need and send the information to Bloomberg. These functions make their work much easier and more convenient.

In details, there is an insert file function that when someone imports a file, it would read the selected data (instrument name, cusip, and maturity months) in the file and used these selected data to do calculations (Cusip and price). The calculations are formulated by the other functions created and after it's done with the calculations, it would populate the data onto the prices page. Lastly, there is a function that submits all the data to Bloomberg. This project is way different from what we did in ScriptEd because there are only functions and no decorations.
